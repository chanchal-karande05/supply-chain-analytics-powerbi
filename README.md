# Supply Chain & Sales Analytics Dashboard (Power BI)

## Project Overview
This project analyzes sales performance and supply chain operations using Power BI.

The dashboard tracks key business metrics such as revenue trends, customer contribution, product demand, and supplier procurement activity.

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling

## Key KPIs
- Total Revenue
- Total Orders
- Total Quantity Sold
- Average Order Value
- Total Customers
- Total Suppliers

## Dashboard Features
- Revenue Trend Analysis
- Top 10 Customer Analysis
- Category Revenue Breakdown
- Supplier Procurement Insights
- Product Demand Analysis
- Executive Insights Page

## Dataset Tables
- sales_orders
- customer_master
- product_master
- supplier_master
- procurement_orders
